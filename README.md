# けものブック
すごーい!けものブックはFacebookの「いいね!」や「うけるね」を「すごーい!」や「たーのしー!」に変えられるChrome拡張なんだね!
Chromeウェブストアからインストールできるんだって!

## へーきへーき! 言語によって対応状況違うから!
けものブックが対応しているのは「にほんご」「にほんご(かんさい)」「えーご」だけだよ!全部の言語に対応するのはちょっとむずかしいんだって!
